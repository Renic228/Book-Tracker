## Описание программы

*"Book Tracker — это GUI‑приложение для учёта прочитанных книг. Программа позволяет: добавлять книги в коллекцию с указанием названия, автора, жанра и количества страниц; просматривать все добавленные книги в виде таблицы;  фильтровать книги по жанру и количеству страниц; сохранять данные в JSON‑файл и загружать их при следующем запуске; проверять корректность ввода данных."*

## Как пользоваться
1.	Заполнрить поля: Название книги, автор, жанр, колличество страниц (Целое положителньое число)
2.	Нажмите кнопку добавить книгу 
4.	Чтобы сбросить фильтры, нажмите кнопку «Сбросить».
5.	Для сохранения данных в файл нажмите «Сохранить в JSON».
6.	Чтобы загрузить ранее сохранённые данные, нажмите «Загрузить из JSON».
________________________________________
## Пример работы
Добавление книги:
Заполните форму:
Нажмите «Добавить книгу».
Книга появится в таблице
Фильтрация книг:
Выберите в фильтре «Жанр» → Фантастика.
Нажмите «Применить фильтр» — в таблице останутся только книги жанра «Фантастика».
Выберите «Страниц >» → 200 и снова нажмите «Применить фильтр» — отобразятся книги жанра «Фантастика» с более чем 200 страницами.
